# Taş-Kağıt-Makas Oyunu

![Taş Kağıt Makas Görseli](https://github.com/umutrenasyuksel/tas-kagit-makas-oyunu/blob/main/tas_kagit_makas.jpg)

## Projenin amacı <br/>
Bu proje, Global AI Hub tarafından düzenlenen Aygaz Python Bootcamp sonucunda final projesi olarak geliştirilmiştir. Bilinen taş kağıt makas oyununu bilgisayara karşı oynamayı sağlar. <br/> 

## Kullanılan Teknoloji <br/>
Python


## Nasıl Oynanır? <br/>
Oyunu oynamak için 'Evet' girilir.  <br/>
Ardından "taş", "kağıt, "makas" seçeneklerinden biri girilir. Bu adımda kullanıcıdan girdi alındıktan sonra bilgisayardan da alınır ve aşağıdaki kurallara göre ilk turun sonucu belirlenir:  <br/>
**Taş, makası yener (Taş, makası kırar).** <br/>
**Kağıt, taşı yener (Kağıt, taşı sarar).** <br/>
**Makas, kağıdı yener (Makas, kağıdı keser).** <br/>
Yenen tarafa 1 puan verilir. Ardından oyun devam eder ve iki taraftan da tekrar "taş, "kağıt, "makas" seçeneklerinden biri alınır.  <br/>
Bu döngü sonucu 2 puana ulaşan oyunu kazanır.  <br/>
Oyun sona erer ve hem kullanıcıya hem de bilgisayara oyunu tekrar oynamak isteyip istemediği sorulur.  <br/>
Verilen cevaplara göre *(evet / hayır)* iki taraf da oynamak isterse oyuna tekrar başlanır. Aksi takdirde oyun sona erer.   <br/>
